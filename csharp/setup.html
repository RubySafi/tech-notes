Preface

In this section, we will explain how to set up the C# environment on VSCode using only extensions licensed under the MIT or similarly permissive licenses.

Installation Steps

1. Install VSCode

Download and install Visual Studio Code (VSCode) from the official website:

Download VSCode

2. Install .NET

Download and install the .NET SDK from the official website:

Download .NET SDK

You can verify if the installation was successful by running the following command: 

dotnet --version

If the installation was successful, the version number will be displayed.

3. Install Extensions

We recommend installing the following extensions for a smooth C# development experience. These extensions are licensed under MIT, but please check the licenses yourself. This site does not guarantee the accuracy of licensing information and assumes no responsibility for any licensing issues. Licenses may change over time, so it is your responsibility to verify the latest terms.

  • Prettier - Code formatter
  • C#
  • .NET Install Tool
  • C# Extensions
  • C# XML Documentation Comments
  • NuGet Package Manager

Next Steps: Customize Your VSCode Environment

Now that you have installed VSCode and the necessary extensions, you can further customize your development environment. This will help you optimize your workflow and improve your coding experience. Below are some recommended customizations.

1. Configure Prettier

Prettier is a popular code formatter that can help you maintain a consistent code style.

2. Set Tab Size and Spacing

By default, VSCode uses 4 spaces for indentation. You can customize this setting based on your preferences. I recommend setting the tab size to 2.
