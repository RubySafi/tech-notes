Enum Files

This page explains how to add an enum file to your project. In this example, we'll assume the project name is SampleProject, the enum file is named SampleEnum.cs, and the enum name is EnumTest.

1. Add a new file to your project and name it SampleEnum.cs.

2. Manually add the following template code:

The recommended template for an enum file is as follows:

SampleEnum.cs 
namespace SampleProject
{
    public enum EnumTest
    {
        Number_A, // 0
        Number_B, // 1
        Number_C  // 2
    }
}

3. Enum Value Notes

By default, the first value in an enum starts at 0, and each subsequent value increments by 1.

If you'd like to start with a different value, you can specify it manually. For example: 

public enum EnumTest
{
    Number_A = 1, // 1
    Number_B,      // 2
    Number_C       // 3
}
