Class Files

This page explains how to add a class file to your project. Unlike Visual Studio, creating a class file in VSCode produces an empty file. Therefore, you'll need to add some template code manually. In this example, we'll assume the project name is SampleProject, and the class file is named SampleClass.cs.

1. Add a new file to your project and name it SampleClass.cs.

2. Manually add the following template code:

The recommended template for a class file is as follows:

SampleClass.cs 
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
using System.Threading.Tasks;

namespace SampleProject
{
    internal class SampleClass
    {
    }
}

Tips

If you are using the "C# Extension", there's an easier way to add a class file. Right-click on the Explorer in VSCode, select "New C# > Class", and follow the prompts.
