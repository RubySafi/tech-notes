Running Without Debugging

In this section, we explain how to build and run without detailed debugging. You can know how your program works if there is no error. It is convenient if you want to know the current UI or briefly check whether your function is properly works. This action is corresponding to Ctrl+F5 in Visual Studio.

Command Line method

Run the following command. 

  dotnet run
