Running Without Debugging

In this section, we explain how to build and run your program without detailed debugging. This is useful for checking how your program behaves when there are no errors. It allows you to quickly see the current UI or verify that a function is working as expected. This action corresponds to pressing Ctrl+F5 in Visual Studio with the "Debug" configuration..

Command Line Method

Run the following command in your terminal:

  dotnet run

Shortcut/Menu Method

You can press Ctrl+F5 to run your program without debugging. Note that the shortcut key may vary depending on your environment or version. Alternatively, you can select Run → Run Without Debugging from the menu bar to achieve the same result.

Tips

You can also confirm the exact shortcut key in the Keyboard Shortcuts settings. To access them, select File → Preferences → Keyboard Shortcuts from the menu.
