TabControl in C# (WinForms)

In C# Windows Forms applications, the TabControl is used to organize content into multiple tabs. Each tab is represented by a TabPage, allowing you to display different sets of controls in a structured way.

Creating a TabControl

The following code demonstrates how to create a TabControl with two tabs:

Form.Designer.cs 

using System.Windows.Forms;
using System.Drawing;

private TabControl tabControl1;
private TabPage tabPage1;
private TabPage tabPage2;
private Button button1;
private Button button2;

private void InitializeComponent()
{
    tabControl1 = new TabControl();
    tabPage1 = new TabPage();
    tabPage2 = new TabPage();
    button1 = new Button();
    button2 = new Button();

    // TabControl properties
    tabControl1.Controls.Add(tabPage1);
    tabControl1.Controls.Add(tabPage2);
    tabControl1.Location = new Point(20, 20);
    tabControl1.SelectedIndex = 0;
    tabControl1.Size = new Size(300, 200);

    // tabPage1
    tabPage1.Controls.Add(button1);
    tabPage1.Location = new Point(4, 29);
    tabPage1.Name = "tabPage1";
    tabPage1.Padding = new Padding(3);
    tabPage1.Size = new Size(447, 306);
    tabPage1.TabIndex = 0;
    tabPage1.Text = "tabPage1";
    tabPage1.UseVisualStyleBackColor = true;

    // tabPage2
    tabPage2.Controls.Add(button2);
    tabPage2.Location = new Point(4, 29);
    tabPage2.Name = "tabPage2";
    tabPage2.Padding = new Padding(3);
    tabPage2.Size = new Size(447, 306);
    tabPage2.TabIndex = 1;
    tabPage2.Text = "tabPage2";
    tabPage2.UseVisualStyleBackColor = true;
    
    // button1
    button1.Location = new Point(6, 6);
    button1.Size = new Size(94, 29);
    button1.Text = "button1";
    button1.Click += button1_Click;
    
    // button2
    button2.Location = new Point(6, 6);
    button2.Size = new Size(94, 29);
    button2.Text = "button2";
    button2.Click += button2_Click;

    // Add the TabControl to the form
    Controls.Add(tabControl1);
}
Form.cs 

private void button1_Click(object sender, EventArgs e)
{
    MessageBox.Show("tab1");
}

private void button2_Click(object sender, EventArgs e)
{
    MessageBox.Show("tab2");
}

Handling Events

The SelectedIndexChanged event can be used to perform actions when the selected tab changes.

Form.Designer.cs 

// Event handler for tab change
tabControl1.SelectedIndexChanged += TabControl1_SelectedIndexChanged;
Form.cs 

private void TabControl1_SelectedIndexChanged(object sender, EventArgs e)
{
    MessageBox.Show("Selected Tab: " + tabControl1.SelectedTab.Text);
}

Adding Controls Dynamically

The TabControl also supports adding new tabs and controls dynamically at runtime. Here's an example: 


// Create a new tab page dynamically
TabPage newTab = new TabPage("New Tab");

// Add controls to the new tab
newTab.Controls.Add(new Button { Text = "Click Me", Location = new Point(20, 20) });

// Add the new tab to the TabControl
tabControl1.TabPages.Add(newTab);

Use Case Example

The TabControl is ideal for organizing complex user interfaces, such as settings pages, multi-step wizards, or categorized content.
