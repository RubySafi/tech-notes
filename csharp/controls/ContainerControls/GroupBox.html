GroupBox Control in C# (WinForms)

In C# Windows Forms applications, the GroupBox control is used to group related controls within a labeled box. It is especially useful for visually organizing controls that share similar functionality or settings.

Creating a GroupBox

The following code demonstrates how to create a GroupBox control with child controls inside:

Form.Designer.cs 

using System.Windows.Forms;
using System.Drawing;

private GroupBox groupBox1;
private RadioButton radioButton1;
private RadioButton radioButton2;

private void InitializeComponent()
{
    groupBox1 = new GroupBox();
    radioButton1 = new RadioButton();
    radioButton2 = new RadioButton();

    // GroupBox properties
    groupBox1.Location = new Point(30, 30);
    groupBox1.Size = new Size(200, 100);
    groupBox1.Text = "Select an Option";

    // RadioButton1 properties
    radioButton1.AutoSize = true;
    radioButton1.Location = new Point(20, 30);
    radioButton1.Text = "Option 1";
    radioButton1.Checked = true;

    // RadioButton2 properties
    radioButton2.AutoSize = true; 
    radioButton2.Location = new Point(20, 60);
    radioButton2.Text = "Option 2";

    // Add radio buttons to the GroupBox
    groupBox1.Controls.Add(radioButton1);
    groupBox1.Controls.Add(radioButton2);

    // Add the GroupBox to the form
    Controls.Add(groupBox1);
}

Common Properties

Here are some commonly used properties for the GroupBox control:

  • Text: Sets the title displayed at the top of the GroupBox.
  • Enabled: Enables or disables all child controls within the GroupBox.
  • BackColor: Changes the background color of the GroupBox.

Example of setting these properties:


groupBox1.Text = "Settings";
groupBox1.Enabled = true;
groupBox1.BackColor = Color.LightYellow;

Use Case Example

The GroupBox control is commonly used to group related options, such as form settings, user preferences, or data input fields. It helps improve readability and organization in complex forms.
