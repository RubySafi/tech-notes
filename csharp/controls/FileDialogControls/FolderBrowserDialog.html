FolderBrowserDialog Control in C# (WinForms)

The FolderBrowserDialog control in C# Windows Forms allows the user to select a folder from the file system. It is useful when you need to prompt the user to choose a folder to save files, open files from, or work with folders in other ways.

Basic Setup

You can create a FolderBrowserDialog instance programmatically, and configure its properties to specify the starting directory and other options.

Form.Designer.cs 

using System;
using System.Windows.Forms;

private FolderBrowserDialog folderBrowserDialog1;
private Button button1;

private void InitializeComponent()
{
    folderBrowserDialog1 = new FolderBrowserDialog();
    button1 = new Button();

    // button1
    // 
    button1.Location = new Point(12, 12);
    button1.Name = "button1";
    button1.Size = new Size(94, 29);
    button1.TabIndex = 0;
    button1.Text = "Select Folder";
    button1.UseVisualStyleBackColor = true;
    button1.Click += button1_Click;
    
    // folderBrowserDialog1
    folderBrowserDialog1.SelectedPath = "C:\\";  // Set the initial folder
    
    Controls.Add(button1);
}
Form.cs 

private void button1_Click(object sender, EventArgs e)
{
    DialogResult result = folderBrowserDialog1.ShowDialog();

    // Check if the user selected a folder
    if (result == DialogResult.OK)
    {
        string folderPath = folderBrowserDialog1.SelectedPath;
        MessageBox.Show($"Folder selected: {folderPath}");
    }
    else
    {
        MessageBox.Show("No folder selected.");
    }
}

Properties

Here are some of the most commonly used properties of the FolderBrowserDialog:

  • SelectedPath: Specifies the initial folder that the dialog will display when opened. For example, "C:\\".
  • Description: Sets a description that appears in the dialog box, guiding the user on what to do. For example, "Select a folder to work with".
  • ShowNewFolderButton: A boolean property that controls whether the "New Folder" button is displayed. Set to true to allow the user to create new folders. The default is true.
  • RootFolder: Defines the root folder that the dialog will start in, such as Environment.SpecialFolder.MyComputer or Environment.SpecialFolder.Desktop.

Tip: The Description property does not change the title bar message! Instead, the message appears just above the input box where the user selects the folder.
