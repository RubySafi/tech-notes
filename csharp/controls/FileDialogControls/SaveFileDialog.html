SaveFileDialog Control in C# (WinForms)

The SaveFileDialog control in C# Windows Forms allows the user to select a file location and specify a file name for saving data. It is typically used when saving documents, images, or other files.

Basic Setup

You can create a SaveFileDialog instance programmatically, and configure its properties to specify filters, initial directory, default file name, and other options.

Form.Designer.cs 

using System;
using System.Windows.Forms;

private SaveFileDialog saveFileDialog1;
private Button button1;

private void InitializeComponent()
{
    saveFileDialog1 = new SaveFileDialog();
    button1 = new Button();

    // button1
    // 
    button1.Location = new Point(12, 12);
    button1.Name = "button1";
    button1.Size = new Size(94, 29);
    button1.TabIndex = 0;
    button1.Text = "Save File";
    button1.UseVisualStyleBackColor = true;
    button1.Click += button1_Click;
    
    // saveFileDialog1
    saveFileDialog1.InitialDirectory = "C:\\";  // Set the initial directory
    saveFileDialog1.Filter = "Text files (*.txt)|*.txt|All files (*.*)|*.*";  // File type filter
    saveFileDialog1.Title = "Save a File";  // Title of the dialog
    saveFileDialog1.DefaultExt = "txt"; // Default file extension
    
    Controls.Add(button1);
}
Form.cs 

private void button1_Click(object sender, EventArgs e)
{
    DialogResult result = saveFileDialog1.ShowDialog();
    
    // Check if the user chose a location to save the file
    if (result == DialogResult.OK)
    {
        string filePath = saveFileDialog1.FileName;
        MessageBox.Show($"File saved at: {filePath}");
    }
    else
    {
        MessageBox.Show("File was not saved.");
    }
}

Properties

Here are some of the most commonly used properties of the SaveFileDialog:

  • InitialDirectory: Specifies the directory that the dialog will display when opened. For example, "C:\\".
  • Filter: Defines the types of files that are visible in the dialog. For example, "Text files (*.txt)|*.txt|All files (*.*)|*.*" allows the user to choose between text files and all file types.
  • Title: Sets the title of the dialog window. For example, "Save a File".
  • FileName: Contains the full path of the file the user selects or enters for saving.
  • DefaultExt: Specifies the default file extension for the saved file. For example, "txt".
  • OverwritePrompt: A boolean property that determines whether the user is prompted before overwriting an existing file. Default is true.
