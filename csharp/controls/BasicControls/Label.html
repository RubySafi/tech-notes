Label Control

In C# Windows Forms applications, the Label control is commonly used to display text or provide instructions to users. Labels are non-interactive, making them ideal for static text content.

Label Creation Template

The following code demonstrates how to create a simple label in your form's designer file:

Form.Designer.cs 

using System.Windows.Forms;
using System.Drawing;

private Label label1;

private void InitializeComponent()
{
    label1 = new Label();

    // Label properties
    label1.AutoSize = true; // Adjusts size automatically based on text
    label1.Location = new Point(20, 20); // Sets position on the form
    label1.Name = "label1";
    label1.TabIndex = 0;
    label1.Text = "Hello, VSCode!"; // Sets the displayed text

    // Add the label to the form
    Controls.Add(this.label1);
}

Common Properties

Here are some commonly used properties for the Label control:

Form.Designer.cs 

label1.BackColor = SystemColors.ActiveCaption; // Background color
label1.BorderStyle = BorderStyle.FixedSingle;   // Adds a visible border
label1.ForeColor = Color.Red;                   // Text color
label1.Visible = false;                         // Hides the label

Practical Tip

The AutoSize property is set to true by default, which automatically adjusts the label's size based on its content. If you prefer to control the size manually, set AutoSize = false and define the Size property.
