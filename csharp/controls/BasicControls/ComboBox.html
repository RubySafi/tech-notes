ComboBox Control in C# (WinForms)

In C# Windows Forms applications, the ComboBox control is used to display a dropdown list of items from which the user can select one. It is useful for situations where you want to limit the user's input to a predefined set of options.

ComboBox Creation Template

The following code shows how to create a ComboBox control and set its properties:

Form.Designer.cs 

using System.Windows.Forms;
using System.Drawing;

private ComboBox comboBox1;

private void InitializeComponent()
{
    comboBox1 = new ComboBox();

    // comboBox1 properties
    comboBox1.DropDownStyle = ComboBoxStyle.DropDownList;  // Prevents user input
    comboBox3.Items.AddRange(new object[] { "foo", "bar", "hoge" });
    comboBox1.Location = new Point(147, 150);
    comboBox1.Name = "comboBox1";
    comboBox1.Size = new Size(121, 28);
    comboBox1.TabIndex = 0;

    // Adding items to the ComboBox
    comboBox1.Items.Add("Option 1");
    comboBox1.Items.Add("Option 2");
    comboBox1.Items.Add("Option 3");

    // Add the ComboBox to the form
    Controls.Add(comboBox1);
}

Common Properties

Here are some commonly used properties for the ComboBox control:

Form.cs 

comboBox1.SelectedIndex = 0;  // Set the selected item by index
comboBox1.SelectedItem = "Option 2";  // Set the selected item by value

Note: These properties should be set in Form.cs, not in Form.Designer.cs.

Handling Events

You can handle the SelectedIndexChanged event to perform an action when the selected item in the ComboBox changes:

Form.Designer.cs 

comboBox1.SelectedIndexChanged += comboBox1_SelectedIndexChanged;
Form.cs 

private void comboBox1_SelectedIndexChanged(object sender, EventArgs e)
{
    // Code to handle the change in selected item
    MessageBox.Show("Selected Item: " + comboBox1.SelectedItem.ToString());
}

Binding an Enum to a ComboBox

It is convenient to use data binding for this purpose. Below is an example of how to bind enum values:

Form.Designer.cs 

this.comboBox1 = new ComboBox();

// Initialize the ComboBox
this.comboBox1.FormattingEnabled = true;
this.comboBox1.Location = new Point(50, 50);
this.comboBox1.Size = new Size(200, 30);

// Add the ComboBox to the form
this.Controls.Add(this.comboBox1);
Form.cs 

// Bind the enum to the ComboBox
comboBox1.DataSource = Enum.GetValues(typeof(DayOfTheWeek));

// Set the default selected value
comboBox1.SelectedItem = DayOfTheWeek.Wed;
Enum.cs 

public enum DayOfTheWeek
{
    Mon,
    Tue,
    Wed,
    Thu,
    Fri,
    Sat,
    Sun
}

Binding a List of Classes to a ComboBox

You can also bind a list of objects to the ComboBox. Below is an example:

Form.Designer.cs 

this.comboBox1 = new ComboBox();

// Initialize the ComboBox
this.comboBox1.FormattingEnabled = true;
this.comboBox1.Location = new Point(50, 50);
this.comboBox1.Size = new Size(200, 30);

// Add the ComboBox to the form
this.Controls.Add(this.comboBox1);
Form.cs 

List classList = new List
{
    new ClassA { Name = "Item1", Id = 1 },
    new ClassA { Name = "Item2", Id = 2 }
};

// Bind the list of ClassA objects to the ComboBox
this.comboBox1.DisplayMember = "Name"; // Display the 'Name' property of ClassA
this.comboBox1.DataSource = classList;

// Set the default selected value
comboBox1.SelectedItem = classList[1];
ClassA.cs 

public class ClassA
{
    public string Name { get; set; }
    public int Id { get; set; }
}
