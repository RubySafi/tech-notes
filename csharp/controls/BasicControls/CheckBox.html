CheckBox Control in C# (WinForms)

In C# Windows Forms applications, the CheckBox control is used to allow the user to select or deselect an option. It provides an easy way to handle boolean-like choices such as "Yes/No" or "True/False."

CheckBox Creation Template

The following code shows how to create and add a CheckBox to your form:

Form.Designer.cs 

using System.Windows.Forms;
using System.Drawing;

private CheckBox checkBox1;

private void InitializeComponent()
{
    checkBox1 = new CheckBox();

    // checkBox1 properties
    checkBox1.AutoSize = true;
    checkBox1.Location = new Point(67, 68);
    checkBox1.Name = "checkBox1";
    checkBox1.Size = new Size(101, 24);
    checkBox1.TabIndex = 0;
    checkBox1.Text = "Enable Option";
    checkBox1.UseVisualStyleBackColor = true;

    // Add the checkbox to the form
    Controls.Add(checkBox1);
}

Common Properties

Here are some commonly used properties for the CheckBox control:

Form.Designer.cs 

checkBox1.Anchor = AnchorStyles.Top | AnchorStyles.Bottom | AnchorStyles.Left | AnchorStyles.Right;
checkBox1.Checked = true;  // Set the checkbox to be initially checked
checkBox1.CheckState = CheckState.Checked;  // Indicate the checkbox is checked

Handling Events

The CheckedChanged event is triggered whenever the state of the checkbox changes. You can handle this event to perform actions based on the user's selection.

The following example demonstrates how to handle the CheckedChanged event:

Form.Designer.cs 

checkBox1.CheckedChanged += checkBox1_CheckedChanged;
Form.cs 

private void checkBox1_CheckedChanged(object sender, EventArgs e)
{
    if (checkBox1.Checked)
    {
        MessageBox.Show("Checkbox is checked.");
    }
    else
    {
        MessageBox.Show("Checkbox is unchecked.");
    }
}

In this example, when the user changes the checkbox state, a message box will appear indicating whether the checkbox is checked or unchecked.
