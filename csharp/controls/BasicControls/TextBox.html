TextBox Control in C# (WinForms)

The TextBox control is a common input field in C# Windows Forms applications. It allows users to enter and edit text. Text boxes are useful for capturing user data such as names, email addresses, and other text-based information.

TextBox Creation Template

The following code demonstrates how to create a simple TextBox in your form's designer file:

Form.Designer.cs 

using System.Windows.Forms;
using System.Drawing;

private TextBox textBox1;

private void InitializeComponent()
{
    textBox1 = new TextBox();

    // TextBox properties
    textBox1.Location = new Point(110, 82); // Position on the form
    textBox1.Name = "textBox1";
    textBox1.Size = new Size(125, 27);      // Width and height
    textBox1.TabIndex = 0;                  // Tab order for keyboard navigation

    // Add the text box to the form
    Controls.Add(textBox1);
}

Common Properties

Here are some commonly used properties for the TextBox control:

Form.Designer.cs 

// Resizes the text box when the form is resized
textBox1.Anchor = AnchorStyles.Top | AnchorStyles.Bottom | AnchorStyles.Left | AnchorStyles.Right;

// Makes the text box read-only (ideal for displaying data)
textBox1.ReadOnly = true;

// Sets placeholder text (available in .NET 6 and later)
textBox1.PlaceholderText = "Enter your name...";

// Masks text input, useful for passwords
textBox1.UseSystemPasswordChar = true;

// Enables multi-line input
textBox1.Multiline = true;

// Adds a scrollbar for longer text
textBox1.ScrollBars = ScrollBars.Vertical;  // Vertical scrollbar only (ideal for paragraphs)

// Alternative scrollbar options
// textBox1.ScrollBars = ScrollBars.Horizontal; // Horizontal scrollbar only (less common)
// textBox1.ScrollBars = ScrollBars.Both;       // Both vertical and horizontal (for wide content)

Practical Tips

  • Use the ReadOnly property instead of Enabled = false if you want to prevent editing while maintaining the default text color.
  • The PlaceHolderText property is supported in .NET 6 and newer versions. For earlier versions, consider adding a label for guidance.
  • For password entry, set UseSystemPasswordChar = true to mask the text.
