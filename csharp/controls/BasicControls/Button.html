Button Control in C# (WinForms)

In C# Windows Forms applications, the Button control is commonly used to trigger actions when clicked. This page explains how to create and configure a button using Visual Studio and C#.

Button Creation Template

The following code demonstrates how to create a simple button in your form's designer file:

Form.Designer.cs 
using System.Windows.Forms;
using System.Drawing;

private Button button1;

private void InitializeComponent()
{
    button1 = new Button();

    // button1 properties
    button1.Location = new Point(10, 10);
    button1.Name = "button1";
    button1.Size = new Size(94, 29);
    button1.TabIndex = 0;
    button1.Text = "Click Me";
    button1.UseVisualStyleBackColor = true;

    // Add the button to the form
    Controls.Add(button1);
}

Common Properties

Here are some commonly used properties for the Button control:

Form.Designer.cs 
button1.Anchor = AnchorStyles.Top | AnchorStyles.Bottom | AnchorStyles.Left | AnchorStyles.Right;
button1.Enabled = false; // Disables the button

Handling Events

When you double-click a button in Visual Studio's Form Designer, the IDE automatically generates an event handler for the Click event. The following code shows the generated code and the corresponding event handler:

Form.Designer.cs 
button1.Click += button1_Click;
Form.cs 
private void button1_Click(object sender, EventArgs e)
{
    MessageBox.Show("Button clicked!");
}

In this example, clicking the button will display a message box with the text "Button clicked!"
