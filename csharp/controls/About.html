Introduction to UI Controls

Adding UI Controls to a Windows Forms Application

In Visual Studio, you can drag and drop UI controls, but in VSCode, you must manually edit the Form.Designer.cs file.

In Visual Studio, some code is automatically generated in the following two sections:

  • Inside the InitializeComponent() method to describe default values for a control.
  • After the row #endregion to declare the field for the control.
Below is an example of how to initialize a Label control in the InitializeComponent() method: 

 #region Windows Form Designer generated code
 private void InitializeComponent()
 { 
     // 
     // label1
     // 
     label1.AutoSize = true;
     label1.Location = new Point(17, 16);
     label1.Name = "label1";
     label1.Size = new Size(116, 20);
     label1.TabIndex = 6;
     label1.Text = "Text";
 }
 #endregion

private Label label1;

On this page, we primarily explain what should be inserted inside the InitializeComponent() method.
