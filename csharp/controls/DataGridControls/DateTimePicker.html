DateTimePicker Control in C# (WinForms)

The DateTimePicker control in C# Windows Forms allows users to select a date and time using a calendar-style interface. This control provides a convenient way to pick dates and times, with options to customize its format and appearance.

Basic Setup

The following example demonstrates how to add a DateTimePicker control to your form and configure its basic properties.

Form.Designer.cs 

using System.Windows.Forms;

private DateTimePicker dateTimePicker1;

private void InitializeComponent()
{
    dateTimePicker1 = new DateTimePicker();

    // Set size and location of the DateTimePicker control
    dateTimePicker1.Location = new Point(12, 12);
    dateTimePicker1.Name = "dateTimePicker1";
    dateTimePicker1.Size = new Size(200, 27);
    dateTimePicker1.TabIndex = 0;

    // Add the DateTimePicker control to the form
    this.Controls.Add(dateTimePicker1);
}

Properties

Below are some frequently used properties for the DateTimePicker control:

  • Value: Gets or sets the current date and time displayed in the control.
  • Format: Sets the format in which the date and time are displayed. Possible values include Long, Short, and Custom.
  • MinDate: Sets the minimum date that can be selected.
  • MaxDate: Sets the maximum date that can be selected.
  • CustomFormat: Defines a custom date and time format string when the format is set to Custom.
  • ShowUpDown: Determines whether the control displays the calendar dropdown or uses up/down arrows to select the date.
