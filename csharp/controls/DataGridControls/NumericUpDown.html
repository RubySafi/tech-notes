NumericUpDown Control in C# (WinForms)

The NumericUpDown control in C# Windows Forms provides a user-friendly way to enter numeric values within a specified range. It features buttons that allow the user to increment or decrement the value, along with input validation to ensure the value stays within a defined minimum and maximum range.

Basic Setup

The following example demonstrates how to add a NumericUpDown control to your form and configure its basic properties.

Form.Designer.cs 

using System.Windows.Forms;

private NumericUpDown numericUpDown1;

private void InitializeComponent()
{
    numericUpDown1 = new NumericUpDown();

    // Set size and location of the NumericUpDown control
    numericUpDown1.Location = new Point(12, 12);
    numericUpDown1.Name = "numericUpDown1";
    numericUpDown1.Size = new Size(150, 27);
    numericUpDown1.TabIndex = 0;
    numericUpDown1.ThousandsSeparator = true;

    // Add the NumericUpDown control to the form
    this.Controls.Add(numericUpDown1);
}

Properties

Below are some frequently used properties for the NumericUpDown control:

  • Minimum: Sets the minimum value that can be selected.
  • Maximum: Sets the maximum value that can be selected.
  • Increment: Specifies the step value when clicking the up or down arrows.
  • Value: Gets or sets the current numeric value in the control.
  • DecimalPlaces: Sets the number of decimal places displayed.
  • ThousandsSeparator: Displays a comma separator for thousands when enabled.
  • ReadOnly: Prevents direct text entry, allowing only changes via the arrow buttons.

Note: The NumericUpDown control's Value property uses the decimal type. This allows for high precision when dealing with fractional values.
