StatusStrip Control in C# (WinForms)

In C# Windows Forms applications, the StatusStrip control is used to display status information at the bottom of the form. It’s often used to provide feedback to users, such as progress updates, status messages, or other details about the application's state.

Creating a StatusStrip

The following code demonstrates how to create a StatusStrip with a label to display a message and a progress bar.

Form.Designer.cs 

using System.Windows.Forms;

private StatusStrip statusStrip1;
private ToolStripStatusLabel toolStripStatusLabel1;
private ToolStripProgressBar toolStripProgressBar1;

private void InitializeComponent()
{
    statusStrip1 = new StatusStrip();
    toolStripStatusLabel1 = new ToolStripStatusLabel("Ready");
    toolStripProgressBar1 = new ToolStripProgressBar();

    // Adding items to the StatusStrip
    statusStrip1.Items.Add(toolStripStatusLabel1);
    statusStrip1.Items.Add(toolStripProgressBar1);

    // Adding the StatusStrip to the form
    Controls.Add(statusStrip1);
}

Adding Different ToolStrip Items

A StatusStrip can also contain various types of items similar to a ToolStrip. Although most ToolStrip items can be added via code, some items — such as ToolStripSeparator — are not compatible with StatusStrip.

The method for adding items is similar to that of a ToolStrip. See the ToolStrip page for details.

ToolStripProgressBar

When using a ToolStripProgressBar, we usually set the following properties: 


toolStripProgressBar1.Value = 50;   // Set the progress bar value to 50%
toolStripProgressBar1.Maximum = 100; // Set the maximum value
toolStripProgressBar1.Minimum = 0;   // Set the minimum value

Use Case Example

The StatusStrip is useful for providing user feedback without disrupting the main interface. It's commonly used in applications that need to show progress updates, such as file transfers or background data processing tasks.
