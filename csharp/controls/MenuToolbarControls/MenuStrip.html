MenuStrip Control in C# (WinForms)

In C# Windows Forms applications, the MenuStrip control is used to create menus, helping you organize commands and options that the user can interact with.

Creating a MenuStrip

The following code demonstrates how to create a MenuStrip with a simple 'File' menu, including a few options. The structure is as follows: 

File(&F)
├── New(&N)
└── Open(&O)
Form.Designer.cs 

using System.Windows.Forms;

private MenuStrip menuStrip1;
private ToolStripMenuItem fileToolStripMenuItem;
private ToolStripMenuItem newToolStripMenuItem;
private ToolStripMenuItem openToolStripMenuItem;

private void InitializeComponent()
{
    menuStrip1 = new MenuStrip();
    fileToolStripMenuItem = new ToolStripMenuItem();
    newToolStripMenuItem = new ToolStripMenuItem();
    openToolStripMenuItem = new ToolStripMenuItem();

    // menuStrip1
    menuStrip1.Items.Add(fileToolStripMenuItem);
    menuStrip1.ImageScalingSize = new Size(20, 20);
    menuStrip1.Location = new Point(0, 0);
    menuStrip1.Name = "menuStrip1";
    menuStrip1.Size = new Size(800, 28);
    menuStrip1.TabIndex = 0;
    menuStrip1.Text = "menuStrip1";

    // fileToolStripMenuItem
    fileToolStripMenuItem.DropDownItems.Add(newToolStripMenuItem);
    fileToolStripMenuItem.DropDownItems.Add(openToolStripMenuItem);
    fileToolStripMenuItem.Name = "fileToolStripMenuItem";
    fileToolStripMenuItem.Size = new Size(63, 24);
    fileToolStripMenuItem.Text = "File(&F)";
    
    // newToolStripMenuItem
    newToolStripMenuItem.Name = "newToolStripMenuItem";
    newToolStripMenuItem.Size = new Size(224, 26);
    newToolStripMenuItem.Text = "New(&N)";

    // openToolStripMenuItem
    openToolStripMenuItem.Name = "openToolStripMenuItem";
    openToolStripMenuItem.Size = new Size(224, 26);
    openToolStripMenuItem.Text = "Open(&O)";
    
    // Add the MenuStrip to the form
    Controls.Add(menuStrip1);
    MainMenuStrip = menuStrip1;
}

Setting Hotkeys

In well-designed applications, hotkeys are commonly used to allow users to quickly navigate the menu. For example, "File(F)" means the user can focus on this menu by pressing the "F" key. To set a hotkey, include an ampersand (&) before the letter, like "File(&F)". This feature is very useful for improving efficiency and accessibility. We strongly recommend setting hotkeys for all menus and submenus. Most operations should be performed without a mouse—only with the keyboard. Without hotkeys, an application can feel cumbersome and inefficient.

More Complex Menu Structures

You can create nested menus by adding DropDownItems. Below is an example of a menu structure with more levels of hierarchy. Detailed menu settings are omitted for brevity. 

File(&F)
├── New(&N)
├── Open(&O)
│   ├── Type1(&2)
│   └── Type2(&3)
└── Exit(&X)
Tool(&T)
└── Option(&O)
Form.Designer.cs 

using System.Windows.Forms;

private MenuStrip menuStrip1;
private ToolStripMenuItem fileToolStripMenuItem;
private ToolStripMenuItem newToolStripMenuItem;
private ToolStripMenuItem openToolStripMenuItem;
private ToolStripMenuItem type1ToolStripMenuItem;
private ToolStripMenuItem type2ToolStripMenuItem;
private ToolStripMenuItem exitToolStripMenuItem;
private ToolStripMenuItem toolToolStripMenuItem;
private ToolStripMenuItem optionToolStripMenuItem;

private void InitializeComponent()
{
  menuStrip1 = new MenuStrip();
  fileToolStripMenuItem = new ToolStripMenuItem();
  newToolStripMenuItem = new ToolStripMenuItem();
  openToolStripMenuItem = new ToolStripMenuItem();
  toolToolStripMenuItem = new ToolStripMenuItem();
  optionToolStripMenuItem = new ToolStripMenuItem();
  type1ToolStripMenuItem = new ToolStripMenuItem();
  type2ToolStripMenuItem = new ToolStripMenuItem();
  exitToolStripMenuItem = new ToolStripMenuItem();
  
  // menuStrip1
  menuStrip1.Items.AddRange(new ToolStripItem[] { fileToolStripMenuItem, toolToolStripMenuItem });
  
  // fileToolStripMenuItem
  fileToolStripMenuItem.DropDownItems.AddRange(new ToolStripItem[] { newToolStripMenuItem, openToolStripMenuItem, exitToolStripMenuItem });
  fileToolStripMenuItem.Text = "File(&F)";
  
  // newToolStripMenuItem
  newToolStripMenuItem.Text = "New(&N)";
  
  // openToolStripMenuItem
  openToolStripMenuItem.DropDownItems.AddRange(new ToolStripItem[] { type1ToolStripMenuItem, type2ToolStripMenuItem });
  openToolStripMenuItem.Text = "Open(&O)";
  
  // toolToolStripMenuItem
  toolToolStripMenuItem.DropDownItems.AddRange(new ToolStripItem[] { optionToolStripMenuItem });
  toolToolStripMenuItem.Text = "Tool(&T)";

  // optionToolStripMenuItem
  optionToolStripMenuItem.Name = "optionToolStripMenuItem";
  optionToolStripMenuItem.Text = "Option(&O)";
  
  // type1ToolStripMenuItem
  type1ToolStripMenuItem.Name = "type1ToolStripMenuItem";
  type1ToolStripMenuItem.Text = "Type1(&1)";
  
  // type2ToolStripMenuItem
  type2ToolStripMenuItem.Name = "type2ToolStripMenuItem";
  type2ToolStripMenuItem.Text = "Type2(&2)";
  
  // exitToolStripMenuItem
  exitToolStripMenuItem.Name = "exitToolStripMenuItem";
  exitToolStripMenuItem.Text = "Exit(&X)";
  
  // Add MenuStrip to form
  Controls.Add(menuStrip1);
  MainMenuStrip = menuStrip1;
}

Different Types of Menu Items

There are several types of items that can be added to a menu, such as separators, comboboxes, and textboxes.

Separator

A separator is used to divide the menu into distinct sections. Below is an example of how to create a separator in a menu.

Form.Designer.cs 

      private ToolStripSeparator toolStripSeparator1;
      
      private void InitializeComponent()
      {
          toolStripSeparator1 = new ToolStripSeparator();
        
          // toolStripSeparator1
          toolStripSeparator1.Name = "toolStripSeparator1";
          toolStripSeparator1.Size = new Size(221, 6);
      }

Grayed (Disabled) Menu Items

Sometimes, you may want to disable a menu item, making it unselectable. This can be done by setting the Enabled property of the menu item to false.

Checkable Menu Items

A checkable menu item can be useful in certain scenarios. You can achieve this by using the following properties:

  • Checked: Determines whether the menu item is checked (e.g., for a checkbox item).
  • CheckOnClick: If set to true, the menu item will be checked when clicked.

Other Types

We do not cover all types of menu items, such as comboboxes or menus that combine multiple techniques. If you're interested in these or need them for implementation, please search for additional resources.

Handling Events

You can handle the Click event of individual ToolStripMenuItem objects to perform actions when a menu item is clicked.

Form.Designer.cs 

// Event handler for the "New" menu item click
newToolStripMenuItem.Click += NewToolStripMenuItem_Click;
Form.cs 

// Event handler for the "New" menu item click
private void NewToolStripMenuItem_Click(object sender, EventArgs e)
{
    MessageBox.Show("New file created!");
}
