ToolStrip Control in C# (WinForms)

The ToolStrip control in C# Windows Forms is used to create a toolbar-like UI element that can hold a variety of controls such as buttons, labels, combo boxes, and other ToolStrip items. It is often used to provide quick access to common actions in an application.

Creating a ToolStrip

Here's how you can create a simple ToolStrip with a few buttons:

Form.Designer.cs 

using System.Windows.Forms;

private ToolStrip toolStrip1;
private ToolStripButton toolStripButton1;
private ToolStripButton toolStripButton2;

private void InitializeComponent()
{
    toolStrip1 = new ToolStrip();
    
    toolStripButton1 = new ToolStripButton();
    toolStripButton1.Name = "toolStripButton1";
    toolStripButton1.Size = new Size(65, 24);
    toolStripButton1.Text = "Button1";
    toolStripButton1.Click += toolStripButton1_Click;
    
    toolStripButton2 = new ToolStripButton();
    toolStripButton2.Name = "toolStripButton2";
    toolStripButton2.Size = new Size(65, 24);
    toolStripButton2.Text = "Button2";
    toolStripButton2.Click += toolStripButton2_Click;

    // Adding buttons to the ToolStrip
    toolStrip1.Items.Add(toolStripButton1);
    toolStrip1.Items.Add(toolStripButton2);
    toolStrip1.Location = new Point(0, 0);
    toolStrip1.Name = "toolStrip1";
    toolStrip1.Size = new Size(800, 27);
    toolStrip1.TabIndex = 0;

    // Adding the ToolStrip to the form
    Controls.Add(toolStrip1);
}
Form.Designer.cs 

private void toolStripButton1_Click(object sender, EventArgs e)
{
    MessageBox.Show("Button 1 clicked!");
}

private void toolStripButton2_Click(object sender, EventArgs e)
{
    MessageBox.Show("Button 2 clicked!");
}

Setting an Icon for ToolStrip Items

Instead of text, you can set an icon for ToolStrip items using the Image property. Here's how you can add an icon to a ToolStripButton: 


toolStripButton1 = new ToolStripButton();
toolStripButton1.Image = Image.FromFile("path\\to\\icon.png");
toolStripButton1.DisplayStyle = ToolStripItemDisplayStyle.Image; 
toolStrip1.Items.Add(toolStripButton1);

The DisplayStyle property ensures that only the icon appears without text. If you want both text and an icon, set DisplayStyle to ToolStripItemDisplayStyle.ImageAndText.

Handling Events

To handle events, such as when a user clicks a button, you can subscribe to the button's Click event. Here's an example of how to handle the click event for a ToolStripButton:

Form.Designer.cs 

// Adding an event handler for ToolStripButton1
toolStripButton1.Click += toolStripButton1_Click;
Form.cs 

// Event handler for ToolStripButton click
private void toolStripButton1_Click(object sender, EventArgs e)
{
    MessageBox.Show("Button 1 clicked!");
}

Adding Different ToolStrip Items

A ToolStrip can contain various types of items, such as buttons, labels, combo boxes, and more. Below are examples of how to add different types of items:

ToolStripButton


ToolStripButton button1 = new ToolStripButton("Click Me");
toolStrip1.Items.Add(button1);

ToolStripLabel


ToolStripLabel label1 = new ToolStripLabel("ToolStrip Label");
toolStrip1.Items.Add(label1);

ToolStripComboBox


ToolStripComboBox comboBox1 = new ToolStripComboBox();
comboBox1.Items.Add("Option 1");
comboBox1.Items.Add("Option 2");
toolStrip1.Items.Add(comboBox1);

ToolStripDropDownButton


ToolStripDropDownButton dropDownButton = new ToolStripDropDownButton("Options");
fooToolStripMenuItem = new ToolStripMenuItem();
barToolStripMenuItem = new ToolStripMenuItem();

fooToolStripMenuItem.Name = "fooToolStripMenuItem";
fooToolStripMenuItem.Size = new Size(224, 26);
fooToolStripMenuItem.Text = "Foo";

barToolStripMenuItem.Name = "barToolStripMenuItem";
barToolStripMenuItem.Size = new Size(224, 26);
barToolStripMenuItem.Text = "Bar";

// Adding items to the DropDownButton
dropDownButton.DropDownItems.AddRange(new ToolStripItem[] 
{ 
    fooToolStripMenuItem, 
    barToolStripMenuItem 
});

// Adding the DropDownButton to the ToolStrip
toolStrip1.Items.Add(dropDownButton);

The ToolStripDropDownButton allows you to create a dropdown menu within the ToolStrip, similar to a MenuStrip. You can nest additional items to create a hierarchical menu structure. For more advanced menu structures, refer to the MenuStrip page.

ToolStrip with ToolStripSeparator

A ToolStripSeparator is useful for visually separating items within the ToolStrip for better organization. Here's how you can add it: 


ToolStripSeparator separator = new ToolStripSeparator();
toolStrip1.Items.Add(separator);

Use Case Example

ToolStrips are commonly used in applications to create toolbars that provide users with easy access to frequently used commands. These toolbars can contain buttons, dropdowns, and other elements that make it easier to interact with the application. ToolStrips can be docked to any edge of the form, making them flexible for different design preferences.
