ToolTip Control in C# (WinForms)

The ToolTip control in C# Windows Forms allows you to display small pop-up text bubbles when the user hovers over a control, such as a button or label. It provides additional information or descriptions for UI elements.

Basic Setup

To use a ToolTip, create an instance of the ToolTip class and associate it with the control you want to display the tip for. The tooltip text is set using the SetToolTip method.

Form.Designer.cs 

using System;
using System.Windows.Forms;

private Button button1;
private ToolTip toolTip1;

private void InitializeComponent()
{
    components = new System.ComponentModel.Container();
    button1 = new Button();
    toolTip1 = new ToolTip(components);
    
    // Set up the ToolTip text for a button
    toolTip1.SetToolTip(button1, "Click to submit the form");

    // button1
    button1.Location = new Point(12, 12);
    button1.Name = "button1";
    button1.Size = new Size(94, 29);
    button1.TabIndex = 0;
    button1.Text = "button1";
    toolTip1.SetToolTip(button1, "Show Dialog");
    button1.UseVisualStyleBackColor = true;
    button1.Click += button1_Click;
}
Form.cs 

private void button1_Click(object sender, EventArgs e)
{
    MessageBox.Show("Hello!");
}

Key Properties of ToolTip

Here are some important properties of the ToolTip control:

  • BackColor: Sets the background color of the tooltip.
  • ForeColor: Sets the text color of the tooltip.
  • IsBalloon: Determines whether the tooltip is displayed in a balloon shape (true or false).
  • ShowAlways: If true, the tooltip is displayed even if the form is not active.
  • AutoPopDelay: Sets the duration (in milliseconds) for how long the tooltip remains visible.
  • InitialDelay: Sets the delay (in milliseconds) before the tooltip is shown after the user hovers over a control.
  • ReshowDelay: Sets the delay (in milliseconds) between subsequent tooltip displays when the user keeps the mouse on a control.

Key Methods of ToolTip

Here are some useful methods for working with the ToolTip control:

  • SetToolTip(Control, string): Sets the tooltip text for a specific control.
  • RemoveAll(): Removes all tooltips associated with controls.
  • Show(string, Point): Manually displays the tooltip at a specific location (optional, typically used for programmatically triggered tooltips).
