Before using Git, you need to configure some settings.

1. Set Your Name

Explanation: Set your name for commits.

git config --global user.name "Your Name"

2. Set Your Email

Explanation: Set your email for commits.

git config --global user.email "your@email.com"

3. Check Configuration

Explanation: View current Git settings.

git config --list

4. Initialize a Repository

Explanation: Create a new Git repository.

git init
