Setup: Installing Git & Configuring GitHub

To start using Git, follow these steps:

1. Install Git

Download and install Git from the official website:

Download Git

After installation, verify it by running the following command:

git --version

2. Configure Git

Set up your username and email:

git config --global user.name "Your Name"
git config --global user.email "your-email@example.com"

Check your settings:

git config --list

3. Create a GitHub Account

Go to GitHub and sign up.

4. Set Up SSH Key (Optional, Recommended)

To avoid entering your password every time, set up an SSH key:

ssh-keygen -t ed25519 -C "your-email@example.com"
